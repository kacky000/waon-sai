# WA音祭（WAON FES）公式Webサイト

和と洋が交錯する、音の祭典。WA音祭の公式Webサイトです。

## 概要

- **イベント名**: WA音祭（WAON FES）
- **開催日**: 202X年7月14日（土）
- **会場**: 新宿HOLIDAY
- **概念**: 「和（日本の和）」×「A（アルファベット/洋楽）」×「音」

## 技術スタック

- HTML5（セマンティックマークアップ）
- CSS3（レスポンシブ対応）
- Vanilla JavaScript（フレームワークなし）

## 機能

### メインサイト
- レスポンシブなシングルページレイアウト
- Hero・Concept・Information・Entry・Artist・Footerの6セクション
- スムーススクロール機能
- スクロールアニメーション

### 管理画面
- ブラウザベースのコンテンツ管理画面
- LocalStorageを使用したデータ永続化
- 以下の項目を編集可能：
  - ヘロー情報
  - コンセプト説明
  - 開催情報（日時、会場、料金）
  - 出演者募集要項
  - アーティスト情報（追加・削除可能）
  - フッター情報

## 使用方法

### メインサイトの表示
```bash
python3 -m http.server 8000
# http://localhost:8000 にアクセス
```

### 管理画面へのアクセス
```
http://localhost:8000/admin.html
```

## 管理画面の操作

1. 左のナビゲーションメニューから編集したいセクションをクリック
2. フォームに内容を入力
3. 「保存」ボタンをクリック
4. 自動的にメインサイトに反映されます

### データの永続化
- すべてのデータはブラウザのLocalStorageに保存されます
- ブラウザを閉じてもデータは消えません
- 「データをリセット」ボタンでデフォルト値に戻すことができます
